# Casa Brasa Barcelona

Carta digital responsive para un restaurante ficticio de cocina mediterránea
contemporánea en Barcelona.

El proyecto forma parte de un portfolio frontend y está orientado a demostrar
composición de componentes, tipado con TypeScript, renderizado de listas,
búsqueda, filtros combinados y gestión de estado local.

## Estado

Proyecto en desarrollo.

La primera etapa prepara la base técnica y documental. La carta, los filtros y
la selección de platos se implementarán en issues posteriores.

## Stack

- React;
- TypeScript;
- Vite;
- Tailwind CSS;
- Lucide React.

## Requisitos

- Node.js 24 o una versión LTS compatible;
- npm.

## Ejecución local

```bash
npm install
npm run dev
```

## Comandos disponibles

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
